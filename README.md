# EthBuildQuest_FadingHope

Fading Hope team, repo for Eth BuildQuest

To setup project
`yarn install`

Auto Compile contract
`npx run watch:compile`

Auto test contract
`npx run watch:test`

Manual compile
`npx run hardhat compile`

Deploy contract
`npx run hardhat deploy`

You need to setup `.env` file before deploy live
